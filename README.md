# Fancy Logo Generator MCP Server

This MCP server generates fancy logos using `gemini-3-pro-image-preview` and `langgraph`.

## Features

- **File Upload:** Upload files from your local drive or Google Drive (when connected to the filesystem MCP or Google Drive MCP).
- **Custom Prompts:** Specify your own prompts for image generation.
- **Prompt Optimization:** Get four AI-powered prompt recommendations to improve your image generation results.
- **Image Configuration:** Configure the image size (e.g., for social media) and the number of images to generate.
- **Image Generation:** Generate images based on your prompts and configurations.
- **Image Upscaling:** Upscale the resolution of your generated images.
- **Multiple Download Formats:** Download your images in PNG, JPEG, or PDF format.

## Deployment

This server is designed to be deployed as a Docker container and listed in the MCP Toolkit Catalog.
